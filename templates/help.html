Guia de Uso del Sistema

Como Empezar
  1. Cargar Configuracion: Subir archivo XML con invernaderos
  2. Seleccionar Invernadero: Elegir invernadero y plan de riego
  3. Ejecutar Simulacion: Ver resultados en tiempo real
  4. Generar Reportes: HTML, XML y graficos TDA
Formato XML

El archivo debe contener:

  • Configuracion de invernaderos
  • Hileras con plantas
  • Drones disponibles
  • Planes de riego predefinidos
Funcionalidades Principales
Simulacion de Drones

Sistema automatizado de riego con drones inteligentes

Reportes Detallados

Estadisticas completas de tiempo, agua y fertilizante

Visualizacion TDA

Graficos de estructuras de datos en tiempo real

Especificaciones Tecnicas
Estructuras de Datos
  • Lista Enlazada: Manejo de plantas e invernaderos
  • Cola (FIFO): Planificacion de riego
  • Pila (LIFO): Estado de drones
Algoritmos
  • Optimizacion: Minimiza tiempo total de riego
  • Distribucion: Asignacion eficiente de recursos
  • Tracking: Seguimiento en tiempo real
Consejos de Uso
Mejores Practicas
  • Verificar formato XML antes de subir archivos
  • Revisar configuracion de invernaderos y drones
  • Generar reportes despues de cada simulacion
  • Utilizar graficos TDA para analizar rendimiento
  • Descargar resultados XML para respaldo
Limitaciones
  • Un solo dron puede regar por vez
  • Tiempo minimo: 1 segundo por planta
  • Archivos XML maximo 10MB
  • Reportes se guardan temporalmente
